# 基于 TypeScript + Vue3 + Vite 技术栈开发的 UI 组件 Demo

1. 安装 `TypeScript`

   ```sh
   $ npm install -g typescript
   # or
   $ yarn global add typescript
   # or
   $ pnpm add -g typescript
   ```

2. 安装 `vite`
   ```sh
   $ yarn global add create-vite-app
   ```

2. 运行项目
   ```sh
   $ yarn dev
   ```
